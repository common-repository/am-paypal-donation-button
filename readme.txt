Drop the AM PayPal donation button plugin on a widget area or insert it in your post or page using shortcode.

== Description ==
Place PayPal donation button on your website in two different ways with minimum setups.
1. Just drop the widget on a widget area, fill in your data and you are ready to accept donations. 
2. Insert PayPal donation button in the post with shortCode. See installation tab.
GOOD LUCK, wish you a lot of donations

== Installation ==
1. Upload the am-PayPal-donation-button folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drop the AM PayPal donation button plugin on a widget area and fill in the required fields
4. To insert donation button in your post open settings page under 'Settings' menu in WordPress.
